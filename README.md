# Ecological risk assessment of microplastic particles in Tokyo Bay sediments

Data and analysis scripts for ecological risk assessment of microplastic particles in Tokyo Bay sediments.

## Associated manuscript

This repository contains the data and analysis scripts associated with the manuscript:

**Preliminary ecological risk assessment of microplastic particles in Tokyo Bay sediments using site-specific power-law extrapolation**

## Overview

This study assessed the preliminary ecological risk of microplastic particles in Tokyo Bay sediments using a three-year sediment monitoring dataset from 36 stations. Measured sediment microplastic concentrations in the 20–5000 µm size range were rescaled to 1–5000 µm using site-specific power-law extrapolation. Ecological risk was assessed by comparing rescaled sediment concentrations with food-dilution-based SSD-derived hazard thresholds using toxicity data aligned with ToMEx 2.0.1.

## Repository structure

- `data/`: Input datasets used in the analysis.
- `scripts/`: R scripts used for power-law rescaling, SSD analysis, and figure generation.
- `outputs/`: Output files, including generated figures and tables.

## Data availability

All data used in this study are available in this repository.
